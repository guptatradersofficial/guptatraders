## Questions & Troubleshooting üÜò

### Common Issues:

**Q: Sitemap not updating?**
A: 
```bash
npm run generate-sitemap
# Or manually: node scripts/generate-sitemap.js
```

**Q: Pages not indexing?**
A: 
1. Check robots.txt allows page
2. Use "Request Indexing" in Search Console
3. Verify page isn't blocked by robots meta

**Q: Structured data errors?**
A:
1. Test with Rich Results Test
2. Fix validation errors
3. Re-test after fixes
4. Wait 24 hours for reprocessing

**Q: Low CTR despite good rankings?**
A:
1. Improve meta description
2. Add rich snippets
3. A/B test titles
4. Improve page speed
